# bo-dynamic-pricing
Official codebase for “Bayesian Optimization for Dynamic Pricing and Learning.” Implements BO-Inf, BO-Fin-Model-Based, and BO-Fin-Heuristic algorithms using Gaussian Processes for demand learning and Bayesian Optimization for adaptive, revenue-maximizing pricing.

Have all the code snippets used in the paper and other comparision algorithms.
